# Cray

Cray 是一个 Rust 实现的 XHTTP+VLESS 协议服务器，专门针对无服务容器场景进行优化，具有低内存开销的特点。

## 诞生背景

Xray-core 在 VPS 这样的传统使用场景下功能固然强大，但在无服务云容器这样的特殊环境中，其不但无法直接通过环境变量进行配置，几十 MB 的内存开销对按使用量计费的云服务来说也是一笔不可忽视的成本。

本项目并不是要为相关领域实现一种新的 core~~（本人能力有限，做不出这种东西）~~，而是旨在优化容器云这种特殊用途。目前正在优先支持 XHTTP+VLESS。

本项目依然处于早期概念阶段，目前只实现了简单的转发功能，欢迎各位大佬参与，但目前所有与容器使用无关的提案、issue、PR均不会被接受。

## 特点

1. 通过环境变量进行配置，无需为容器环境单独编写启动脚本。
2. 极低内存开销，通常不会超过 10MB。
